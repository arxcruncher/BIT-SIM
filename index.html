What is BIT-SIM?
BIT-SIM is an open source Bittorrent simulator and implemented according the original bittorrent protocol specification and simulates the application level behaviour of bittorrent. BIT-SIM uses the OMNeT++ simulation framework to operate.

BIT-SIM is being developed at the Dept. of Telecommunication Systems of Blekinge Institute of Technology, Sweden.
Ongoing Research
The BTH media distribution lab engages in research on minimising inter-AS load, while maintaining high enough QoS to be able to deliver streaming HD video. The main techniques for this are various kinds of mesh-based load sharing and partial caching.

Version 1.1 (release date 09.05.04)
The simulator can be downloaded in different compression formats. See the included INSTALL file for installation directives. BIT-SIM is a simulator software packet that is run on top of OMNeT++. Hence, OMNeT++ must be installed prior. BIT-SIM is open source and distributed under the GNU licence.

The API of the BIT-SIM is available online but can also be generated locally with doxygen. A Doxyfile is found in the main directory of every distribution of the simulator that will pass directives to doxygen.
